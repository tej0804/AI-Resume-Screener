# AI Resume Screening & Job Matching System

An end-to-end AI system that matches resumes to job descriptions using:
- Rule-based skill & experience matching
- TF-IDF lexical similarity
- SBERT semantic similarity

## Features
- Resume preprocessing & skill extraction
- Job description scraping (Internshala)
- Hybrid matching & ranking
- Explainable scoring

## Tech Stack
- Python
- Pandas, Scikit-learn
- Sentence-BERT
- NLP (TF-IDF, cosine similarity)
